# Rejuuv videos

Videos are loaded from this folder for **form-check** (plan screen) and **Day 1 Win** (default: `hip_mobility.mp4`).

**Current local videos (by `exercise_key`):**  
`bridge`, `calf_stretch`, `clamshells`, `heel_slide`, `hip_flexor_stretch`, `hip_mobility`, `prone_press_ups`, `quad_stretch`, `repeated_extension`, `scapular_setting`, `short_arc_quad`, `shoulder_rotation`, `sleeper_stretch`, `straight_leg_raise`.

- **Day 1 Win:** Add `day1-win.mp4` here, or set `EXPO_PUBLIC_DAY1_WIN_VIDEO_URL` to a remote URL. If unset, the app uses `hip_mobility.mp4` from this folder.
- **Body-area-specific Day 1 (optional):** `day1-win-neck.mp4`, `day1-win-lower_back.mp4`, etc. (not yet wired; currently one default is used).
